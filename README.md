# 🔐 PrivacyVault — 隐私保险箱

> 端到端加密文件管理与隐私相册，纯客户端运行，数据永不上传服务器。

## ✨ 功能

- **🔑 密码保护** — PBKDF2 + AES-256-CBC 端到端加密
- **📂 文件保险箱** — 拖拽上传任意文件，一键批量加密/解密
- **🖼️ 隐私相册** — 照片自动加密存储，网格缩略图 + 全屏预览
- **🎨 高端交互** — 玻璃态暗色UI、动态光球背景、流畅动画
- **💯 纯前端** — 无需服务器，所有数据存储在浏览器 IndexedDB 中

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可使用。

```bash
git clone https://github.com/YOUR_USER/privacy-vault.git
cd privacy-vault
# 用浏览器打开 index.html
```

## 🛡️ 安全架构

| 层级 | 方案 |
|------|------|
| 密钥派生 | PBKDF2 (100,000 轮 SHA-256) |
| 加密算法 | AES-256-CBC + 随机 128-bit IV |
| 数据存储 | IndexedDB (浏览器本地) |
| 密码修改 | 全量文件自动重加密 |

## 📸 截图

首次使用设置主密码 → 进入保险箱 → 拖放文件加密 → 隐私相册管理照片。

## ⚠️ 注意

- 密码不会上传到任何服务器
- **忘记密码 = 数据永久丢失**，无法恢复
- 清除浏览器数据会删除所有加密文件，请提前导出
